## Location Privacy

- **Current Best Practices for Generalizing Sensitive Species Occurrence Data**
  GBIF Secretariat (2024)
  https://docs.gbif.org/sensitive-species-best-practices/master/en/

  Key guidance on determining species sensitivity using a four-criteria framework, recommended obscuration levels (0.001° to 0.1°), and Darwin Core field usage for documenting data generalizations. Recommends grid-based generalization over randomization.
